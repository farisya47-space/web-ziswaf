# Madarijus Salikin - Sistem Keuangan ZISWAF

Aplikasi manajemen keuangan untuk Yayasan Madarijus Salikin menggunakan Vue 3 + JavaScript + Tailwind CSS.

## Tech Stack

- Vue 3 (Composition API)
- Vue Router
- Pinia (State Management)
- Tailwind CSS
- Chart.js + vue-chartjs
- Flatpickr
- Lucide Icons
- jsPDF + html2canvas

## Setup Project

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

### 3. Build for Production

```bash
npm run build
```

## Fitur

- ✅ Login & Authentication
- ✅ Dashboard dengan statistik dan charts
- ✅ Manajemen Kategori (ZISWAF)
- ✅ Transaksi Pemasukan
- ✅ Transaksi Pengeluaran
- ✅ Laporan Keuangan dengan Export PDF
- ✅ Profil Pengguna
